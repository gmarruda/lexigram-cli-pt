# lexigram-cli

## Usage
Login to Amazon:
``` bash
lexigram login
```

Download config file:
``` bash
lexigram init-config
```

Generate Lambda function source file:
``` bash
lexigram generate-function [skill]
```

Deploy skill to Amazon
``` bash
lexigram deploy [skill]
```

## Options
[skill]: kazni or koko (default: kazni)
